# Practice Projects

A collection of small JavaScript practice projects focused on backend development, and learning web development fundamentals.

## Projects

### 1. Fetch API with PokeAPI
A demonstration project using the Fetch API to retrieve and display Pokémon data from the PokeAPI. Built to practice asynchronous JavaScript and working with RESTful APIs.

- **Location**: `FetchApiPractice/`
- **Files**: `FetchApiPractice.html`, `index.js`
- **How to run**: Open `FetchApiPractice.html` in your browser
- **API**: [PokeAPI](https://pokeapi.co/) (no key required)

### 2. Weather Forecast App (Open-Meteo API)
An interactive weather application that fetches 7-day forecast data based on user-provided coordinates. Features comprehensive error handling for invalid inputs, network failures, and API errors.

- **Location**: `SecondFetchApiPractice/`
- **Files**: `weather.html`, `index.js`
- **How to run**: Open `weather.html` in your browser
- **API**: [Open-Meteo API](https://open-meteo.com/) (no key required)
- **Features**: Input validation, async/await patterns, error handling, weather code interpretation
- **Date**: February 11, 2026

## Repository Structure

- `FetchApiPractice/` — Pokémon data fetch demo (HTML + JS)
- `SecondFetchApiPractice/` — Weather forecast app with coordinate input (HTML + JS)
